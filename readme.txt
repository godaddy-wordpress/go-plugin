Provides layouts for the CoBlocks Layout Selector.

== Description ==

This plugin adds layouts for the new CoBlocks Layout Selector, to make building pages much easier and faster.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/go` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Create a new page and select a layout to start with.

== Frequently Asked Questions ==

= Does this plugin require Gutenberg? =
Yes. The layouts provided by the Go plugin will not display unless you're running the Gutenberg plugin, or WordPress 5.5+

= Does this plugin require CoBlocks? =
Yes. CoBlocks provides the mechanism for adding layouts to the page while the Go plugin provides those layouts.
